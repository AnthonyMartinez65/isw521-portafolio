- Manipulación del DOM

- Eventos
- addEventListener
- Tipos de eventos

## Lo que aprendí

Durante esta clase aprendí que JavaScript permite agregar interactividad a las páginas web mediante la manipulación del DOM.

También comprendí cómo crear, eliminar, reemplazar y mover elementos utilizando métodos como appendChild(), remove() y replaceWith().

Además, conocí el uso de addEventListener para responder a eventos generados por el usuario, como clics, escritura en formularios y movimientos del mouse.
