# Actividad

Durante esta clase participé en la demostración de Tailwind CSS.

Practicamos la creación de interfaces utilizando clases utilitarias, agregamos efectos `hover`, realizamos un diseño responsivo con `md:` y analizamos las ventajas de Tailwind frente al CSS tradicional y Bootstrap.

Esta actividad me permitió comprender cómo desarrollar interfaces de forma más rápida y mantener un sistema de diseño consistente.
