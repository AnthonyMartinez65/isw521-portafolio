# Clase 05 - Bootstrap y Tailwind CSS

## Temas vistos

- Bootstrap
- Tailwind CSS
- Utility-First
- Mobile-First
- Design System
- Ventajas y desventajas de Tailwind

## Lo que aprendí

Durante esta clase aprendí las diferencias entre Bootstrap y Tailwind CSS.

Bootstrap ofrece componentes ya diseñados, mientras que Tailwind utiliza un enfoque Utility-First, donde cada clase cumple una función específica.

También comprendí cómo Tailwind facilita el diseño responsivo mediante prefijos como `md:` y `lg:`, y cómo permite construir interfaces sin escribir hojas de estilo tradicionales para cada componente.
