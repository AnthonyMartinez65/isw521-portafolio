# Glosario - ISW-521 Programación en Ambiente Web I

Este glosario reúne conceptos importantes vistos durante la primera semana del curso, relacionados con fundamentos de la Web, redes, URLs, DNS y GitHub.

---

## Internet

Red global de computadoras interconectadas que se comunican mediante el protocolo TCP/IP. Es la infraestructura sobre la que funcionan distintos servicios como la Web, el correo electrónico, FTP, SSH y otros.

---

## World Wide Web

Servicio de información que funciona sobre Internet. Permite acceder a documentos, páginas y recursos enlazados mediante HTTP o HTTPS.

---

## TCP/IP

Conjunto de protocolos que permite la comunicación entre dispositivos conectados a una red. Es la base técnica de Internet.

---

## HTTP

Protocolo usado para la comunicación entre clientes y servidores web. Permite solicitar y recibir recursos como páginas HTML, imágenes, archivos JSON, entre otros.

---

## HTTPS

Versión segura de HTTP. Usa cifrado mediante TLS para proteger la comunicación entre el cliente y el servidor.

---

## Cliente

Dispositivo, aplicación o programa que realiza una solicitud a un servidor. Por ejemplo, un navegador web, una aplicación móvil o una herramienta como Postman.

---

## Servidor

Sistema que recibe solicitudes de los clientes, las procesa y devuelve una respuesta. Puede entregar páginas web, datos, imágenes, archivos o servicios.

---

## Modelo Cliente-Servidor

Modelo de comunicación donde un cliente solicita recursos y un servidor los procesa y responde. Es una de las bases principales del funcionamiento de las aplicaciones web.

---

## Web 1.0

Primera etapa de la Web. Se caracterizaba por páginas estáticas donde el usuario principalmente leía información, sin mucha interacción.

---

## Web 2.0

Etapa de la Web donde los usuarios pueden interactuar, crear contenido, comentar, compartir información y participar en plataformas digitales.

---

## Web 3.0

Evolución de la Web relacionada con datos semánticos, descentralización, blockchain, contratos inteligentes y aplicaciones distribuidas.

---

## W3C

World Wide Web Consortium. Organización encargada de definir estándares relacionados con la Web, como HTML, CSS, SVG, accesibilidad web y otros.

---

## IETF

Internet Engineering Task Force. Organización encargada de definir protocolos técnicos que hacen funcionar Internet, como HTTP, TLS, DNS, TCP/IP, SMTP y SSH.

---

## Internet pública

Red abierta y descentralizada a la que pueden acceder usuarios desde cualquier parte del mundo, siempre que tengan conexión y permisos para entrar al recurso solicitado.

---

## Intranet

Red privada usada dentro de una organización. Utiliza tecnologías como HTTP, DNS y TCP/IP, pero su acceso está restringido a empleados o miembros internos.

---

## Extranet

Extensión controlada de una intranet hacia usuarios externos autorizados, como socios, proveedores o clientes. No es pública como Internet, pero tampoco es totalmente interna.

---

## Firewall

Sistema de seguridad que filtra el tráfico de red. Permite o bloquea conexiones según reglas definidas por puerto, dirección IP, protocolo u otros criterios.

---

## VPN

Virtual Private Network. Tecnología que crea un túnel cifrado para acceder de forma segura a una red privada desde una ubicación externa.

---

## URI

Uniform Resource Identifier. Identificador general de un recurso. Puede identificar un recurso por nombre, ubicación o ambos.

---

## URL

Uniform Resource Locator. Tipo de URI que indica la ubicación de un recurso y cómo acceder a él.
